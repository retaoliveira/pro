Usando Git e Github

Definições Git é um sistema de controle de versão por linha de comando. Se você não estiver familiarizado com ele, Git não tem botões para pressionar, apenas comandos para digitar para controlá-lo, mas ele permite salvar múltiplas versões de um arquivo. A beleza do controle de versão é que se você fizer “alguma besteira”, você pode voltar para a última versão salva. É como chegar a um “ponto de salvamento” em um videogame no qual você pode voltar se seu personagem morrer.
